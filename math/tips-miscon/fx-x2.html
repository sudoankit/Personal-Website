Is $f(x) = x^2$ a function?

Answer The statement $f(x) = x^2$ is not a function definition; it's an equality statement assumed to hold for all $x$ in the domain of $f$. The domain and co-domain have to be specified.

To understand clearly, quickly recall the definition of a function:

A function, $f$, is a mathematical object that maps or relates elements of two sets, the domain $A$ to the one called the co-domain $B$ and for $f$ be a function $f : A \to B$, $f$ assigns to each element of $A$ to exactly one element of $B$.

I'm not fond of this quite washed out definition. I like this one found in set theory: $$f = \{ (a,b) \in A \times B : b=f(a) \} \subseteq A \times B$$
